## Define Data Science and Machine Learning

*	Data Science: The field of collecting, analyzing, and interpreting data to gain insights. 
*	Machine Learning: A branch of Data Science that enables computers to learn from data and make predictions. 

### The relationship between:
Machine Learning is a tool used within Data Science.
Example: In house price prediction, Data Science prepares the data, while Machine Learning predicts the house price.

## Data Science Lifecycle
*	Problem Definition 
*	Data Collection 
*	Data Preparation 
*	Exploratory Data Analysis (EDA) 
*	Model Building 
*	Model Evaluation 
*	Deployment 
* Machine Learning fits in: Model Building and Model Evaluation because it is used to train and test predictive models

## Supervised vs. Unsupervised Learning
* Uses labeled data	Uses unlabeled data
* Predicts outcomes	Finds hidden patterns
* Example (Supervised): Predicting house prices.
* Example (Unsupervised): Grouping customers based on shopping behavior.

 ## What Causes Overfitting?
## Causes:
* Complex models 
* Small datasets 
* Too much training 
## Prevention:
* Use more data 
* Simplify the model 
* Apply regularization 
* Use cross-validation 

## Training Data vs. Test Data
*	Training Data: Used to train the model. 
*	Test Data: Used to evaluate the model. 
Common Split: 80% training and 20% testing.
Why Necessary? To measure how well the model performs on new, unseen data and avoid overfitting.

## Case Study: Healthcare
Study: Deep Learning for Diabetic Retinopathy Detection (Gulshan et al., 2016).
Summary: Researchers trained a deep learning model to detect diabetic retinopathy from eye images. The model achieved accuracy similar to medical specialists.
Findings: AI can help doctors diagnose eye diseases faster and more accurately.
### Data Science Lifecycle Stages Covered:
* Data Collection 
* Data Preparation 
* Model Building 
*	Model Evaluation 
*	Deployment (healthcare screening systems)
